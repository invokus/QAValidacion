# Proyecto de Pruebas de API con Karate

## Prerequisitos

Asegúrate de tener las siguientes dependencias y configuraciones en tu máquina local:

- IDE: IntelliJ versión 2024.1.3
- Maven: versión 3.8.6 (debe estar en la variable de entorno)
- JDK: versión 1.8 (debe estar en la variable de entorno)
